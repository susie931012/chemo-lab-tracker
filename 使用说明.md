# 化疗指标记录本地网页

## 打开方式

直接双击 `index.html` 即可使用。数据会保存在当前浏览器本地，不会上传到网络。

如果浏览器限制本地文件功能，也可以在本文件夹中运行：

```powershell
python -m http.server 8765
```

然后打开：

```text
http://127.0.0.1:8765/
```

## 第一版包含的功能

- 手动录入每次检测的指标结果
- 手动新增、修改、删除本次指标行
- 自定义添加、单个删除、批量删除常用指标
- 指标库中的已有指标可单独编辑名称、单位、最低值、最高值
- 指标库可维护默认单位、参考最低值、参考最高值
- 可把勾选的指标保存为固定指标类，例如“血常规”
- 本次录入可选择某个指标类，一键带入该类指标
- 校对表格中的指标可通过下拉框选择
- 可从指标库点击“录入”，把某个指标加入本次记录后手动填写结果
- 可勾选多个指标后点击“选中加入本次”，批量生成录入项
- 可点击“全部加入本次”，一次生成整套手动录入表
- 每次录入时自动带入单位和参考范围
- 保存时根据结果和参考范围自动判断“正常 / 偏高 / 偏低”
- 查看单个指标的折线趋势图
- 添加发现病情、初诊、手术、化疗开始时间、疗程/周期、用药、升白针、发热等治疗事件
- 历史记录可展开查看每次检测的全部指标详情
- 历史记录和治疗事件可编辑，保存后“完整情况”同步更新
- “完整情况”按化疗周期汇总所有治疗事件、用药和每次指标详情
- 导出 CSV，方便用 Excel 打开或带给医生看
- 导出/导入完整备份，方便换电脑或备份数据
- 每个模块都可以独立折叠/展开，也可以一键全部展开或全部折叠
- 支持 Firebase 在线同步，家人可用同一家庭代码协作记录
- 清空当前浏览器中的本机数据

## 在线同步

当前版本已接入 Firebase。使用前请在 Firebase 控制台确认：

1. Authentication 已开启“电子邮件/密码”登录方式。
2. Firestore Database 已创建。
3. 网页能访问互联网。

使用方式：

1. 在“在线同步”模块输入邮箱、密码、家庭代码。
2. 第一次使用点击“注册”，之后点击“登录”。
3. 家人使用同一个家庭代码登录，即可读写同一份数据。

建议家庭代码使用不容易被猜到的名字，例如 `family-2026-xxxx`。医疗信息比较敏感，不建议使用公开或过短的家庭代码。

Firestore 规则建议限制为登录用户才能读写：

```text
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /families/{familyId}/app/main {
      allow read, write: if request.auth != null;
    }
  }
}
```

这版规则适合家庭内部先用。后续如果需要更严格，可以再做“家庭成员白名单”。

## 换电脑或备份

这个网页没有外部依赖，复制整个 `chemo-lab-tracker` 文件夹到其他电脑即可打开使用。

但录入的数据保存在浏览器本地，不会自动跟着文件夹走。迁移数据请按这个流程：

1. 在旧电脑点击“导出完整备份”，得到一个 `.json` 文件。
2. 把 `chemo-lab-tracker` 文件夹和这个 `.json` 备份文件一起复制到新电脑。
3. 在新电脑打开 `index.html`。
4. 点击“导入备份”，选择 `.json` 文件。

“导出 CSV”只适合给医生或 Excel 查看指标记录；完整迁移请使用“导出完整备份”。

## 录入方式

当前版本不再提供报告图片/PDF识别，只保留更稳定的本地录入方式。

推荐流程：先维护指标库，也可以把常用指标保存成“指标类”。录入时点击“录入”、“选中加入本次”、“全部加入本次”，或选择某个指标类后点击“加入该类指标”，再逐项填写结果。

## 自定义指标

在“指标库”中输入指标名称，例如 `总胆红素 TBIL`，也可以填写单位、最低值、最高值，再点击“添加指标”。如果输入的是已有指标，会更新这个指标的单位和参考范围。

指标库列表中的每一行都可以直接修改名称、单位、最低值、最高值，修改后点击这一行的“保存”。

之后有三种手动录入方式：

1. 点击某个指标旁边的“录入”，把它加入本次记录。
2. 勾选多个指标后点击“选中加入本次”。
3. 点击“全部加入本次”，一次加入指标库里的所有指标。
4. 点击“新增一行”，再从指标下拉框中选择。

本次录入时只需要填写“结果”。单位和参考范围会从指标库自动带入，状态会根据结果和参考范围自动判断。勾选多个指标后点击“删除选中”，可以批量清理指标库；删除指标库内容不会删除已经保存的历史检测记录。

## 指标类

在指标库中勾选多个指标，输入指标类名称，例如 `血常规`，点击“用选中指标保存为一类”。

之后在“本次录入”中选择这个指标类，点击“加入该类指标”，就会自动带入这一组指标。比如可以把红细胞、白细胞、血小板等保存为“血常规”。

## 编辑历史

“历史记录”里可以展开每次记录，直接修改日期、报告类型、指标结果、单位和参考范围，点击“保存”后会重新计算状态，并同步更新趋势图和“完整情况”。

“治疗事件”列表也可以直接修改日期、开始时间、疗程/周期、事件名称、用药和备注，点击“保存”后“完整情况”会同步更新。

## 完整情况

“治疗事件”可以记录事件类型、日期、开始时间、疗程/周期、事件名称、用药情况和备注。

“历史记录”中的每次检测都可以展开查看全部指标。“完整情况”会先按化疗开始事件生成周期汇总，每个周期内按时间升序混合列出治疗事件和当期指标记录；同一天优先显示指标记录，再显示复诊、用药等事件。下方完整时间线按日期合并圆点，同一天的多个事件或检查会在同一个圆点下竖向显示名称。悬浮圆点可看摘要，点击圆点可查看当天所有指标、用药和备注详情。

## 示例文本

```text
白细胞 3.1 10^9/L 3.5-9.5 偏低
血红蛋白 108 g/L 115-150 偏低

CEA 8.2 ng/mL 0-5 偏高
ALT 42 U/L 7-40 偏高
肌酐 66 umol/L 45-84 正常
```
